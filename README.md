# Stone Paper Scissors Game 🎮

A two-player Stone Paper Scissors game built in Python.

## Project Structure
## How to Run
```bash
python main.py
```

## How to Play
- Enter names for Player 1 and Player 2
- Enter `1` to play a round
- Enter `0` to see final scores and exit
- Each round both players get a random choice
- Highest score at the end wins!

## Rules
- Stone beats Scissors
- Scissors beats Paper
- Paper beats Stone

## Tech Used
- Python 3
- Modular programming
- Random module
